# Dice-Rolling-Game
A terminal-based dice rolling game built using Python basics and loops.

## Features
- Roll two dice simultaneously.
- Generates new random numbers with every roll.
- Asks the user if they want to roll again.

## Concepts Used
- **Basic Functions & Inputs:** Captures and processes user input.
- **Modules:** Uses the `random` module to generate random values for each die.
- **Loops:** A `while` loop allows continuous rolling until the user chooses to exit.

## How to Run the Script
1. Make sure you have **Python 3.x** installed on your machine.
2. Clone or download this script to your workspace.
3. Open your terminal or command prompt, navigate to the folder, and run:

```bash
python dice_game.py
